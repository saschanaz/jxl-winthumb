# jxl-winthumb

A JPEG XL (*.jxl) WIC decoder to render thumbnails on Windows File Explorer or view images on any WIC-capable image viewers.

## Build environment

Use the stable Rust toolchain. Current toolchain as of 23th July 2023 is 1.71.0.
